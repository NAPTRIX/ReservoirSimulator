# Advanced Reservoir Simulator

A web-based application for modeling and analyzing petroleum reservoir behavior.

## Demo

**[Live Application](https://naptrix.github.io/ReservoirSimulator/)**

## Overview

Interactive reservoir simulation tool designed for engineers and researchers to model oil and gas reservoir dynamics directly in the browser.

## Features

- Interactive web-based interface
- Real-time simulation and visualization
- Comprehensive reservoir modeling tools
- No installation required

## Local Development

```bash
git clone https://github.com/NAPTRIX/ReservoirSimulator.git
cd ReservoirSimulator
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
